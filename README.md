# Telemost Audio Recorder

Бот для записи аудио из Яндекс Телемоста с использованием Playwright.

## Описание

Node.js бот, который автоматически подключается к Яндекс Телемосту через браузер, захватывает аудиопоток с микрофона и сохраняет его в формате WAV для дальнейшей транскрибации.

##  Основные возможности

- Автоматическое подключение к конференциям Яндекс Телемоста
- Захват аудиопотока через системный микрофон
- Запись аудио в формате WAV с помощью FFmpeg
- Гибкая настройка параметров записи
- Логирование процесса работы

## Технологический стек

- **Playwright** - автоматизация браузера
- **FFmpeg** - обработка и запись аудио
- **Node.js** - среда выполнения
- **Winston** - система логирования
